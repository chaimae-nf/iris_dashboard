
 Dash Multi-Page Application – Iris Dataset Analysis

Cette application web interactive, développée avec **Dash** et **Plotly**, permet une exploration approfondie du célèbre dataset **Iris** à travers plusieurs pages thématiques, combinant visualisations graphiques et analyses dynamiques.

### Fonctionnalités principales :

* **Tableau interactif :**
  Visualisation complète du dataset Iris avec possibilité de filtrer et trier les données. Les colonnes incluent `sepal_length`, `sepal_width`, `petal_length`, `petal_width`, `species` et `species_id`.

* **Histogramme dynamique :**
  Affichage de la distribution des longueurs de sépales (`sepal_length`) avec un contrôle interactif du nombre de bins via un slider.

* **Scatter Plot interactif :**
  Nuage de points comparant `sepal_length` et `petal_length`. Un menu déroulant permet de filtrer par espèce (`setosa`, `versicolor`, `virginica`) pour une analyse ciblée.

### Technologies utilisées :

* **Python** pour le traitement des données et la logique de l’application.
* **Dash** pour le développement de l’interface web multi-pages.
* **Plotly** pour des visualisations graphiques interactives et professionnelles.
* **Pandas** pour la manipulation et l’analyse des données.

### Objectif :

Créer une application web professionnelle démontrant la capacité à développer des **dashboards interactifs multi-pages** avec filtrage dynamique, visualisations graphiques et présentation claire des données. Ce projet illustre également l’intégration de bonnes pratiques pour la création de visualisations analytiques et l’exploration de données en Python.
